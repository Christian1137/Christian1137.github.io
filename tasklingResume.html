Description

Developed with a team a full-stack productivity web application using MongoDB, Express.js, React, and Node.js where users earn currency for completing tasks and can purchase virtual items for a digital pet. Implemented secure user authentication, password reset functionality using the SendGrid API, and RESTful backend services for task and reward management.
